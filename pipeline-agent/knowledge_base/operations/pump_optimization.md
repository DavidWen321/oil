# 泵站运行优化指南

## 1. 泵组优化原则

### 1.1 基本原则

1. **安全性原则**: 确保管道各点压力在允许范围内
2. **经济性原则**: 在满足输量的前提下，最小化能耗
3. **可靠性原则**: 优先选择运行稳定的泵组合
4. **均衡性原则**: 各泵运行时间尽量均衡，延长设备寿命

### 1.2 优化目标

$$\min \sum_{i=1}^{n} N_i = \min \sum_{i=1}^{n} \frac{\rho g Q_i H_i}{\eta_i}$$

约束条件：
- 各站进站压力 ≥ 最小允许进站压力
- 各站出站压力 ≤ 最大允许出站压力
- 管道末端压力 ≥ 最小终点压力

## 2. 泵组合选择策略

### 2.1 常用泵型配置

以ZMI系列泵为例：

| 泵型号 | 额定流量 (m³/h) | 额定扬程 (m) | 效率 (%) |
|-------|----------------|-------------|---------|
| ZMI480 | 480 | 280 | 82 |
| ZMI375 | 375 | 220 | 80 |
| ZMI320 | 320 | 180 | 78 |

### 2.2 组合方式

**单泵运行**:
- 适用于低流量工况
- 能耗最低但流量受限

**双泵并联**:
- 同型号并联：流量加倍，扬程不变
- 不同型号并联：需考虑工作点匹配

**多泵串联**:
- 扬程叠加，流量不变
- 适用于长距离、高扬程需求

### 2.3 最优组合判定流程

```
1. 根据输量需求，确定候选泵组合
2. 对每个组合进行水力计算
3. 检查压力约束是否满足
4. 计算总能耗
5. 选择能耗最低的可行方案
```

## 3. 变频调节

### 3.1 变频调节原理

泵的相似定律：
$$\frac{Q_1}{Q_2} = \frac{n_1}{n_2}$$
$$\frac{H_1}{H_2} = \left(\frac{n_1}{n_2}\right)^2$$
$$\frac{N_1}{N_2} = \left(\frac{n_1}{n_2}\right)^3$$

### 3.2 变频节能效果

变频调节相比阀门调节的节能量：
$$\Delta N = N_0 \left[1 - \left(\frac{Q}{Q_0}\right)^3\right]$$

| 流量比 (Q/Q₀) | 节能比例 |
|--------------|---------|
| 0.9 | 27% |
| 0.8 | 49% |
| 0.7 | 66% |
| 0.6 | 78% |

### 3.3 变频运行注意事项

1. **频率下限**: 一般不低于额定频率的60%
2. **散热问题**: 低频运行时注意电机散热
3. **共振区域**: 避开机组共振频率区间

## 4. 运行工况调整

### 4.1 季节性调整

| 季节 | 特点 | 调整策略 |
|-----|-----|---------|
| 夏季 | 粘度低，流动性好 | 可适当降低出站压力 |
| 冬季 | 粘度高，流动阻力大 | 增加出站压力或启用加热 |
| 过渡期 | 工况变化频繁 | 加强监控，及时调整 |

### 4.2 输量变化调整

| 输量变化 | 调整措施 |
|---------|---------|
| 减量10%以内 | 变频调节 |
| 减量10~30% | 减少运行泵台数 |
| 减量30%以上 | 考虑越站运行 |
| 增量 | 增加运行泵或提高频率 |

### 4.3 故障工况处理

- **单泵故障**: 启用备用泵，调整运行方案
- **站内全停**: 启动越站阀，保持管道畅通
- **紧急减压**: 开启泄压阀，防止超压

## 5. 能耗分析

### 5.1 能耗构成

| 项目 | 占比 | 优化空间 |
|-----|-----|---------|
| 输送能耗 | 60~70% | 中等 |
| 加热能耗 | 20~30% | 较大 |
| 辅助能耗 | 5~10% | 较小 |

### 5.2 能耗指标计算

**单位输量能耗**:
$$E_q = \frac{W}{Q \cdot L} \times 10^6$$

单位：kWh/(千吨·公里)

**泵站效率**:
$$\eta_{station} = \frac{\rho g Q \Delta H}{1000 \times W_{elec}}$$

### 5.3 节能措施

1. 优化泵组合，提高运行效率
2. 采用变频调节替代阀门调节
3. 合理安排输油计划，减少启停次数
4. 定期维护设备，保持良好状态
5. 利用夜间低谷电价输油

## 6. 智能优化算法

### 6.1 遗传算法

适用于大规模组合优化问题，可同时优化多个泵站的运行方案。

### 6.2 动态规划

适用于多站联动优化，考虑站间耦合关系。

### 6.3 机器学习方法

利用历史运行数据，建立能耗预测模型，指导实时优化。

## 7. ZMI480 / ZMI375 泵型参数与建模口径

为了保证优化模型与现场一致，推荐统一使用以下参数口径：

| 参数 | ZMI480 | ZMI375 | 说明 |
|---|---:|---:|---|
| 典型单泵扬程 | 230~260 m | 170~210 m | 实际以试验曲线或台账为准 |
| 典型排量区间 | 420~520 m³/h | 320~420 m³/h | 与频率、介质粘度相关 |

## 8. 组合枚举策略（最多 4×4）

在单站组合优化中，可将 `ZMI480` 与 `ZMI375` 的启泵台数分别限制在 0~4 台，形成最多 25 组候选方案：

```
for n480 in [0..4]:
  for n375 in [0..4]:
    if n480 == 0 and n375 == 0:
      continue
    计算总扬程、末站压力、功率、年费用
```

对于每个候选方案，执行以下步骤：

1. 计算总可提供扬程：`H_total = n480 * H480 + n375 * H375 + H_inlet`
2. 计算管道所需扬程：`H_req = H_friction + Δz + H_end_required`
3. 计算平衡误差：`ΔH = H_total - H_req`
4. 若约束满足，则进入经济性计算并打分

## 9. 优化目标与约束

### 9.1 常见目标函数

目标 A：最小化末站压力偏差

$$
\min f_1 = |P_{end} - P_{target}|
$$

目标 B：最小化综合能耗/电费

$$
\min f_2 = \frac{\rho g Q H}{\eta_p \eta_m} \times T \times Price
$$

工程中常采用“先满足安全，再按费用最优”的分层策略。

### 9.2 必要约束

- 末站压力约束：`P_end > 0`（或高于最小安全阈值）
- 设备约束：单泵启停次数、最小连续运行时间
- 电气约束：站内容量、电机温升、变频器频率下限
- 工艺约束：高黏原油工况下不得低于最低流速

## 10. 方案评估指标与HITL建议

推荐在方案对比表中至少给出以下指标：

1. 年耗电量（kWh/a）
2. 年电费（元/a）
3. 末站压力安全裕度（MPa）
4. 方案复杂度（是否涉及频繁启停）

当多个方案电费差异小于 2% 时，建议进入 HITL（人工确认）流程，由调度员结合设备状态、检修窗口、负荷波动预期做最终决策，避免“模型最优但运行不稳”。
