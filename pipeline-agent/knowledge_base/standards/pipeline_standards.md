# 管道工程标准与规范

## 1. 设计标准

### 1.1 管道设计压力

根据《输油管道工程设计规范》(GB 50253)：

| 管道等级 | 设计压力范围 (MPa) |
|---------|-------------------|
| 低压管道 | ≤ 1.6 |
| 中压管道 | 1.6 ~ 4.0 |
| 高压管道 | 4.0 ~ 10.0 |
| 超高压管道 | > 10.0 |

### 1.2 管道流速限制

- **最小流速**: 0.5 m/s（防止蜡沉积）
- **经济流速**: 1.0 ~ 2.0 m/s
- **最大流速**: 3.0 m/s（防止管道冲蚀）

### 1.3 管壁厚度计算

$$\delta = \frac{PD}{2[\sigma]\phi} + C$$

其中：
- δ: 计算壁厚（mm）
- P: 设计压力（MPa）
- D: 管道外径（mm）
- [σ]: 许用应力（MPa）
- φ: 焊缝系数（0.6~1.0）
- C: 腐蚀裕量（一般取1~2mm）

## 2. 运行标准

### 2.1 泵站进出站压力要求

| 参数 | 最小值 | 推荐值 | 最大值 |
|-----|-------|-------|-------|
| 进站压力 | 0.2 MPa | 0.3~0.5 MPa | 0.8 MPa |
| 出站压力 | - | 取决于设计 | 设计压力 |
| 进出站压差 | - | 尽量小 | 设计允许值 |

### 2.2 原油含水率标准

| 等级 | 含水率 (%) | 处理要求 |
|-----|-----------|---------|
| 一级 | ≤ 0.5 | 可直接输送 |
| 二级 | 0.5 ~ 2.0 | 需脱水处理 |
| 三级 | > 2.0 | 必须脱水 |

### 2.3 温度控制

- **最低输送温度**: 高于凝点3~5°C
- **最高输送温度**: 不超过90°C
- **加热站间距**: 根据热力计算确定

## 3. 安全标准

### 3.1 安全系数

| 项目 | 安全系数 |
|-----|---------|
| 管道设计安全系数 | ≥ 1.5 |
| 泵站设计安全系数 | ≥ 1.25 |
| 阀门压力等级 | ≥ 1.5 × 设计压力 |

### 3.2 保护措施

- **超压保护**: 设置安全阀，动作压力为设计压力的1.05~1.1倍
- **低压保护**: 设置低压切断，动作压力一般为0.15~0.2 MPa
- **越站功能**: 每座泵站应具备越站能力

## 4. 能效标准

### 4.1 泵效率要求

| 泵类型 | 最低效率 | 推荐效率 |
|-------|---------|---------|
| 离心泵 | ≥ 75% | ≥ 80% |
| 螺杆泵 | ≥ 70% | ≥ 75% |
| 齿轮泵 | ≥ 65% | ≥ 70% |

### 4.2 电机效率要求

根据《电动机能效限定值及能效等级》(GB 18613)：

| 能效等级 | 效率要求 |
|---------|---------|
| 一级（高效） | ≥ 95% |
| 二级（节能） | ≥ 92% |
| 三级（普通） | ≥ 89% |

### 4.3 输油单耗指标

| 管道类型 | 单耗指标 (kWh/千吨公里) |
|---------|----------------------|
| 大口径长距离 | ≤ 3.5 |
| 中等口径 | ≤ 5.0 |
| 小口径支线 | ≤ 8.0 |

## 5. 维护标准

### 5.1 检测周期

| 检测项目 | 周期 |
|---------|-----|
| 管道壁厚检测 | 每年1次 |
| 防腐层检测 | 每3年1次 |
| 内腐蚀检测 | 每5年1次 |
| 智能清管 | 每年1~2次 |

### 5.2 维护保养

- **泵组维护**: 每500~1000运行小时检查一次
- **阀门维护**: 每季度活动一次
- **仪表校验**: 每年校验一次

## 6. 关键国标与行标应用要点

### 6.1 GB 50251-2015（输气管道）可借鉴条款

尽管该标准主要面向输气工程，但其中关于线路选择、站场布置、安全间距与完整性管理的原则，可在输油场景中作为风险校核参考：

1. 线路选线优先避开高地质灾害区与密集建设区；
2. 阀室、站场及穿越段需考虑检修通道和应急可达性；
3. 设计阶段应同步考虑运行监测与泄漏预警条件。

### 6.2 GB 50253-2014（输油管道）核心执行点

输油工程设计、施工、试运与验收的核心依据应落在该规范。实际执行时建议关注：

- 管道强度与稳定性计算是否覆盖高温、低温、腐蚀裕量等工况；
- 站场工艺流程是否支持调峰、越站与应急切换；
- 保护系统（超压、低压、联锁停泵）是否形成闭环。

### 6.3 SY/T 5536 水力计算规范落地要点

该规范强调参数口径统一与工况边界清晰。实际项目中常见问题是“数据齐全但口径不一致”，导致模型输出偏差。建议：

1. 明确所有输入参数的单位体系；
2. 对历史统计参数给出数据来源与时间范围；
3. 对关键结果（压降、末站压力、泵功率）实施双算校核。

## 7. Barlow公式与壁厚设计补充

在工程初算阶段，常使用 Barlow 公式快速校核承压能力：

$$P = \frac{2St}{D}$$

其中 `P` 为允许内压、`S` 为材料许用应力、`t` 为壁厚、`D` 为外径。结合腐蚀裕量和制造偏差后，才能得到工程壁厚选型值。对于老旧管道改造，不应直接套用原始设计壁厚，必须叠加腐蚀减薄检测数据。

## 8. 泵站间距与材质选择建议

### 8.1 泵站间距

泵站间距不是固定常数，而是“地形高差 + 摩阻损失 + 调峰策略 + 经济性”共同决定。一般可通过以下原则初选：

- 平缓地形、口径较大时可适当拉大间距；
- 山区与高差突变段应缩短间距并提高调节能力；
- 优化目标若偏向节能，通常倾向于降低单站过高扬程。

### 8.2 管材与防腐

材质选择应综合考虑设计压力、环境腐蚀、施工难度与生命周期成本。常见策略包括：

1. 主干线采用高强钢等级并配套外防腐层；
2. 土壤腐蚀区或杂散电流区加装阴极保护；
3. 运行阶段结合内检测（ILI）结果动态调整检维修计划。

## 9. 规范符合性清单（建议用于报告）

为便于在 AI 报告中生成“可审计结论”，建议构建以下符合性清单：

- 设计压力与壁厚计算依据是否完整；
- 关键设备（泵、电机、阀门）选型是否满足能效等级要求；
- 运行参数是否长期落在许可区间；
- 保护与联锁策略是否定期验证；
- 检测、维护、应急演练记录是否完备。

该清单可直接转化为报告中的“风险提示 + 整改建议”结构，支持管理层快速决策与闭环追踪。
