# 管道工程标准与规范

## 1. 设计标准

### 1.1 管道设计压力

根据《输油管道工程设计规范》(GB 50253)：

| 管道等级 | 设计压力范围 (MPa) |
|---------|-------------------|
| 低压管道 | ≤ 1.6 |
| 中压管道 | 1.6 ~ 4.0 |
| 高压管道 | 4.0 ~ 10.0 |
| 超高压管道 | > 10.0 |

### 1.2 管道流速限制

- **最小流速**: 0.5 m/s（防止蜡沉积）
- **经济流速**: 1.0 ~ 2.0 m/s
- **最大流速**: 3.0 m/s（防止管道冲蚀）

### 1.3 管壁厚度计算

$$\delta = \frac{PD}{2[\sigma]\phi} + C$$

其中：
- δ: 计算壁厚（mm）
- P: 设计压力（MPa）
- D: 管道外径（mm）
- [σ]: 许用应力（MPa）
- φ: 焊缝系数（0.6~1.0）
- C: 腐蚀裕量（一般取1~2mm）

## 2. 运行标准

### 2.1 泵站进出站压力要求

| 参数 | 最小值 | 推荐值 | 最大值 |
|-----|-------|-------|-------|
| 进站压力 | 0.2 MPa | 0.3~0.5 MPa | 0.8 MPa |
| 出站压力 | - | 取决于设计 | 设计压力 |
| 进出站压差 | - | 尽量小 | 设计允许值 |

### 2.2 原油含水率标准

| 等级 | 含水率 (%) | 处理要求 |
|-----|-----------|---------|
| 一级 | ≤ 0.5 | 可直接输送 |
| 二级 | 0.5 ~ 2.0 | 需脱水处理 |
| 三级 | > 2.0 | 必须脱水 |

### 2.3 温度控制

- **最低输送温度**: 高于凝点3~5°C
- **最高输送温度**: 不超过90°C
- **加热站间距**: 根据热力计算确定

## 3. 安全标准

### 3.1 安全系数

| 项目 | 安全系数 |
|-----|---------|
| 管道设计安全系数 | ≥ 1.5 |
| 泵站设计安全系数 | ≥ 1.25 |
| 阀门压力等级 | ≥ 1.5 × 设计压力 |

### 3.2 保护措施

- **超压保护**: 设置安全阀，动作压力为设计压力的1.05~1.1倍
- **低压保护**: 设置低压切断，动作压力一般为0.15~0.2 MPa
- **越站功能**: 每座泵站应具备越站能力

## 4. 能效标准

### 4.1 泵效率要求

| 泵类型 | 最低效率 | 推荐效率 |
|-------|---------|---------|
| 离心泵 | ≥ 75% | ≥ 80% |
| 螺杆泵 | ≥ 70% | ≥ 75% |
| 齿轮泵 | ≥ 65% | ≥ 70% |

### 4.2 电机效率要求

根据《电动机能效限定值及能效等级》(GB 18613)：

| 能效等级 | 效率要求 |
|---------|---------|
| 一级（高效） | ≥ 95% |
| 二级（节能） | ≥ 92% |
| 三级（普通） | ≥ 89% |

### 4.3 输油单耗指标

| 管道类型 | 单耗指标 (kWh/千吨公里) |
|---------|----------------------|
| 大口径长距离 | ≤ 3.5 |
| 中等口径 | ≤ 5.0 |
| 小口径支线 | ≤ 8.0 |

## 5. 维护标准

### 5.1 检测周期

| 检测项目 | 周期 |
|---------|-----|
| 管道壁厚检测 | 每年1次 |
| 防腐层检测 | 每3年1次 |
| 内腐蚀检测 | 每5年1次 |
| 智能清管 | 每年1~2次 |

### 5.2 维护保养

- **泵组维护**: 每500~1000运行小时检查一次
- **阀门维护**: 每季度活动一次
- **仪表校验**: 每年校验一次
